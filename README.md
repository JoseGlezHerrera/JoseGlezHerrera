## 👨‍💻 About me

I write code and I attack code. Those two halves feed each other: I build with a security mindset because I know what I'd try if I were on the other side — and I audit better because I know how software actually gets built under a deadline.

- 🍅 **Running in production right now:** [**Pedidos Merca**](https://github.com/JoseGlezHerrera/pedidos-merca) — a system that turns free-form WhatsApp messages into printable invoices for a wholesale market stall in Gran Canaria. **Used every single day**, on a 1 GB VM, with 2 runtime dependencies.
- 🔐 **Security side:** I've worked hands-on with **Huntress** (MDR), **ThreatLocker** (Zero Trust), **Kaseya VSA**, **BitDefender** and **IT Glue** during an internship at an Irish MSP in Dublin.
- 🎓 **CFGS DAM** (2024–2026) and **CFGM SMR** (2022–2024) — both completed **while working full-time**.
- 🧪 I'd rather ship one thing that survives real users than ten CRUD demos. Every project here has a story about what broke in production.
- 🌍 Spanish (native) · English (professional) — open to remote work and relocation.
